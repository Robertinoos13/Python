# Script Name
Alarm with a GUI

## Description
This script is also a GUI for an alarm, which has options like any regular alarm

## Output
The script produces a GUI interface, perfect for a basic alarm

## Libraries
To run this script, you need the following Python libraries:
- 'tkinter'
- 'pygame'

## Main Script Features
- Count seconds and minutes
- Can playing a sound when the alarm is ready (optional)
- Costum message when the alarm is ready
- The user can set a costum seconds number for counting
- Repeating Mode

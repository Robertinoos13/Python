# Script Name
Teams generator

## Description
This script is a random teams generator: you input a number of players and the program put this inputed players in 2 different teams,up to 20 inputed players

## Output
The script produces 2 different teams with different inputed players

## Libraries
To run this script, you need the following Python libraries:
- 'random'
- 'time'

## Main Script Features
- More options to input more or less number of players
- Equal number of players on every team

# Script Name
Auto Clicker

## Description
This script is a automatic autoclicker

## Output
The script produces automatic left clicks on a specified location using coordonates X and Y

## Libraries
To run this script, you need the following Python libraries:
- 'time'
- 'pyautogui'

## Main Script Features
- Can auto clicks on your screen
- 5 seconds to get ready
- You can put pause of some seconds between clicks

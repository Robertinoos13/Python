# Script Name
Random Password Generator

## Description
This script is a Password Generator, where you can generate a random password if you dont have idea what password you can use for your account, phone/laptop security password to open it, ect.

## Output
The script produces a random generated string

## Libraries
To run this script, you need the following Python libraries:
- 'random'

## Main Script Features
- You can spam program runs to generate more passwords faster
- Easy to use
- random.choice() function example
- Don't generate very long passwords,only the minimum of most password systems condition

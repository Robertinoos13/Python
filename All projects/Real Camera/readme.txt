# Script Name
Real Camera

## Description
This script is a simple script what opens your camera and see how other peoples sees you

## Output
The script produces a window and shows the your MAIN CAMERA POV from your laptop/PC, on a flipped mode, like how peoples sees you

## Libraries
To run this script, you need the following Python libraries:
- 'numpy'
- 'cv2'

## Main Script Features
- Can opens your camera
- A new window appears in your screen
- Flipped POV, so like this sees you the peoples
